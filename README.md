# life-dashboard

- Mac mini + RaspberryPi で人生に必要なメトリクス・情報を取得するための Prometheus+Grafana セットです。
- 慣れれば1時間かからないくらいで必要なものが揃うようになっています。
- GitHub Actions の self-hosted ランナーによってデプロイされますので、自動でCDが実行できます。
